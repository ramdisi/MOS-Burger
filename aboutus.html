About Us

Welcome to MOS Burger POS – a modern, web-based Point of Sale system developed with simplicity, speed, and usability in mind.

Our Purpose

MOS Burger POS was created as a personal project to showcase practical frontend development skills using HTML, CSS, and JavaScript. It is designed to meet the needs of fast food outlets, burger shops, and small-scale restaurants that require a lightweight and responsive POS solution.

What We Offer
  • Role-based login system for Admins and Cashiers
  • Add Food Items under 6 catogories
  • Fast billing interface for cashiers
  • Real-time receipt generation and download to the device
  • Generating monthly report and Top Users Report

Technology Used

This project is built entirely with:

  • HTML for structure
  • CSS and Bootstrap layout and design
  • Vanilla JavaScript for application logic
  • Chart js for charts and html2pdf For generate bills & Reports
